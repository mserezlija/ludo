# Covjece, ne ljuti se (LUDO)
Implementacija klasicne drustvene igre za 1-4 igraca.

### Kontrole u igri
#### Ljudski igraci

- Enter za bacanje kockice
- 1-4  unos broja za odabir figure koju zelite pomaknuti
- 1 - izvlacenje nove figure iz baze (kad dobijete 6)
- 2 - pomicanje postojece figure (kad dobijete 6)

#### Racunalni igraci
Racunalo automatski igra svoje poteze prema pravilima.
